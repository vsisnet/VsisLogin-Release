## 🇬🇧 English

### About

VsisLogin is a desktop application for managing multiple browser profiles with advanced fingerprint spoofing, proxy management, browser extension control, and Android emulator integration.

### Features

| Module | Description |
|--------|-------------|
| **Profile Manager** | Create & manage unlimited browser profiles with unique fingerprints |
| **Fingerprint Spoofing** | User-Agent, Screen Resolution, CPU, RAM, GPU, WebGL, Canvas, AudioContext, ClientRects, Fonts |
| **Proxy Manager** | HTTP & SOCKS5 proxy support with live validation and IP geolocation |
| **Extension Manager** | Download from Chrome Web Store, upload local CRX files, assign to profiles |
| **Group Profiles** | Organize profiles into groups/folders |
| **Bulk Operations** | Create, delete, move, configure multiple profiles at once |
| **Emulator Manager** | Create, clone, configure Android emulators with proxy and APK install |
| **Auto Updates** | Automatic version checking and update notifications |

### Installation

1. Go to [**Releases**](https://github.com/vsisnet/VsisLogin-Release/releases/latest)
2. Download `VsisLogin_v4.3.0.rar`
3. Extract and run `VsisLogin.exe`

### System Requirements

- Windows 10/11 (64-bit)
- .NET 8.0 Runtime
